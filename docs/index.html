Pockets

Save and organize your links with ease. A simple Chrome extension to help you save links for later.

Quick Save

Save any link with just a right-click using the context menu.

Easy Access

Access your saved links anytime from the extension popup.

Secure Storage

Your links are stored in a secure storage via silo API.

Notifications

Get notified when links are saved successfully.

Get started by installing the extension from the Chrome Web Store.

Install Extension

To view the links you've saved, you can use this application available at: Link Hub. You can host it yourself or use the hosted version.
Or visit your installed extension links page by clicking "View All Links" in the extension popup.
For more information how to setup the extension and the app, visit the Guide [How to use].
